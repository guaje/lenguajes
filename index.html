## Variables
## Variables: Ejemplos en C++ ```cpp #include <iostream> using namespace std; bool esPar(int x) { return x % 2 == 0; } bool esVocal(char v) { return v == 'a' || v == 'e' || v == 'i' || v == 'o' || v == 'u'; } int main() { bool miBooleano = true; bool miBooleanoEnNumero = 1; char miCaracter = 'A'; char miCaracterEnASCII = 65; int miEntero = 5; float miFlotante = 3.0; char miArreglo[] = {'a', 'e', 'i', 'o', 'u'}; int miMatriz[2][2] = {{255, 0}, {0, 255}}; return 0; } ```
## Variables: Ejemplos en Python ```python def esPar(x): return x % 2 == 0 def esVocal(v): return v == 'a' or v == 'e' or v == 'i' or v == 'o' or v == 'u' miBooleano = True miBooleanoEnNumero = bool(1) miCaracter = 'A' miCaracterEnASCII = chr(65) miEntero = 5 miFlotante = 3.0 miLista = ['a', 'e', 'i', 'o', 'u'] miTupla = ('a', 'e', 'i', 'o', 'u') miMatriz = [[255, 0], [0, 255]] ```
## Variables: Definición __Variable__: Es un espacio de memoria donde se almacena un dato, un espacio donde se guarda la información necesaria para realizar las acciones que ejecutan los programas.

Variables: Declaración

  • Tipo de variable *
  • Nombre o identificador de la variable

De la siguiente manera \(\mathbb{T} x;\) **

Donde \(\mathbb{T}\) es el tipo de dato o conjunto al que pertenece la variable y \(x\) es el identificador o nombre de la variable.

* En lenguajes como python no es necesario declarar el tipo, esto es porque python posee tipado dinámico.
** En lenguajes como pyhton no es necesario el caracter ';'.

## Variables: Recomendaciones Una buena práctica de programación es asignarle el nombre a una variable de tal manera que indique por un lado el papel que desempeña dicha variable en el algoritmo y por otro los posibles valores que almacena.
## Comentarios
## Comentarios: Ejemplos en C++ ```cpp #include <iostream> using namespace std; /* Comentario multi-linea * Este predicado evalua si un numero * es par. */ bool esPar(int x) { return x % 2 == 0; } int main() { // Comentario de una linea cout << esPar(2) << endl; // cout y endl son parte de la libreria iostream cout << esPar(5) << endl; cout << esPar(44) << endl; return 0; } ```
## Comentarios: Ejemplos en Python ```python ''' Comentario multi-linea Este predicado evalua si un numero es par. ''' def esPar(x): return x % 2 == 0 # Comentario de una linea print esPar(2) # print siempre imprime en una linea nueva print esPar(5) print esPar(44) ```
